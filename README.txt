# Multi-Asset Portfolio Dashboard

Este projeto é um dashboard interativo desenvolvido com Dash + Plotly para análise de portfólios compostos por até 5 ativos financeiros (ações, ETFs, criptomoedas).

## Funcionalidades

- Seleção de período histórico (30d, 90d, 1y, 5y, 10y, etc)
- Escolha de até 5 ativos com alocação personalizada
- Cálculo de métricas financeiras: CAGR, Volatilidade, Sharpe
- Gráficos base 100 com exibição opcional dos componentes
- Exportação dos dados em CSV e do gráfico em PNG
- Layout responsivo e intuitivo

pip install -r requirements.txt
